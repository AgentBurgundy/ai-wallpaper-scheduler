# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-XX

### Added
- Initial release
- AI image generation using Google Nano Banana Pro API
- Windows desktop background automation
- Daily scheduling support (Task Scheduler + built-in scheduler)
- Interactive setup script (`npm run setup`)
- Comprehensive logging with Winston
- Environment-based configuration
- Open source MIT license
- Contributing guidelines
- GitHub Actions CI/CD workflows

### Features
- Generate AI images from text prompts
- Automatically set Windows wallpaper
- Run once or schedule daily updates
- Secure API key management via `.env` file
- Configurable image prompts and settings
